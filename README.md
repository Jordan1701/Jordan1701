## Featured Projects

### [Housing Affordability in San Diego County](https://github.com/jordanmitchell/housing-affordability-san-diego)
*Dual ML project predicting housing affordability by ZIP code across San Diego County*

Combines Zillow home value data, Redfin market metrics, and U.S. Census ACS income data to model affordability at the ZIP-code level — a problem with direct relevance to mortgage lending strategy, CRA performance context, and member financial health at community financial institutions. Built two supervised learning pipelines: a regression model predicting a continuous affordability index and a binary classifier distinguishing affordable vs. unaffordable ZIP codes. Findings quantify the coastal premium, income-to-price imbalance, and key market drivers of affordability deterioration across the county.

`Python` `scikit-learn` `Pandas` `Zillow` `Census ACS` `Redfin` `GeoPandas` `Matplotlib`
